# How to run these scripts
Clone CLAASP repo https://github.com/Crypto-TII/claasp

Create the docker enviroment (`docker compose -f docker-compose.yml create environment`)

Start the docker and run `python3 script.py &` or `sage script.sage &`

Press Ctrl + P, Ctrl + Q to detach and keep the computation running.

Press Ctrl + D to exit and stop the container (this will terminate all running scripts).
