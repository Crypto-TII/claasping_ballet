# CLAASP Symmetric Cipher Analysis of [Ballet](http://www.jcr.cacrnet.org.cn/EN/10.13868/j.cnki.jcr.000335)

This repository contains all the scripts used to produce the results presented
in the research paper "More Brisés in Ballet: Extending Differential and Linear Cryptanalysis", eprint: [https://eprint.iacr.org/2026/501](https://eprint.iacr.org/2026/501)

Prerequisite:
Before running the experiments, install the CLAASP library, version 3.2.0.
You can find it at the following link: https://github.com/Crypto-TII/claasp.
You can install it directly on your machine or use the Docker image.
